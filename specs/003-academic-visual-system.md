# Spec 003: Restrained academic visual system

## Required behavior

- Use a white canvas, charcoal text and one claret accent, with a quiet blue secondary only where semantic distinction is needed.
- Use system serif and sans-serif stacks; do not depend on remote web fonts.
- Use no gradients, decorative blobs, animation, hover lift, oversized marketing hero or nested cards.
- Cards are reserved for repeated records and use a radius of 4px or less.
- Use a readable text measure, stable responsive grids, visible focus states and a mobile navigation control.
- Real research imagery may be shown with captions. Generated scientific evidence or stock-like atmosphere imagery is prohibited.

## Acceptance criteria

- Desktop widths 1440px and 1024px and mobile widths 390px and 360px have no horizontal scrolling or overlap.
- Navigation is keyboard-accessible and usable without JavaScript on desktop; mobile menu remains accessible.
- Long titles and URLs wrap without resizing their parent.
- Lighthouse-relevant basics are present: metadata, semantic landmarks, alt text, reduced motion and adequate contrast.
